YARN
===============
Gerenciador de Pacotes Node.js = **Evolução do NPM!**

Instalação
------------------
- Instalar o Yarn:
```bash
$ npm install -g yarn
```
-----------------
- Iniciar um projeto com o `Package.json`:
```bash
$ yarn init -y
```
- Instalar e salvar uma dependência (ex: axios):
```bash
$ yarn add axios
```
- Remover uma dependência (ex: axios):
```bash
$ yarn remove axios
```
- Atualizar uma dependência (ex: axios):
```bash
$ yarn update axios
```
